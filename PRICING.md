# Ценообразование aineron.ru (июль 2026)

Рублёвый биллинг: баланс в копейках, цена за сообщение — фиксированная (flat),
независимо от длины контекста. Применение цен:

```bash
docker-compose exec web python manage.py add_laozhang_models   # цены моделей
docker-compose exec web python manage.py setup_tariffs         # тарифная сетка
```

## Маржа (консервативно: официальные цены API, 80 ₽/$, диалог ~3k in + 600 out токенов)

| Модель | Себестоимость | Цена | Маржа |
|---|---|---|---|
| GPT-4o | ~1,1 ₽ | 4 ₽ | ×3,6 |
| GPT-5 | ~0,8 ₽ | 12 ₽ | ×15 |
| Claude Opus 4.8 | ~7 ₽ | 20 ₽ | ×2,8 |
| Claude Haiku 4.5 | ~0,5 ₽ | 1,50 ₽ | ×3 |
| DeepSeek V3 | ~0,1 ₽ | 1 ₽ | ×10 |
| DALL-E 3 | ~3,2 ₽ | 8 ₽ | ×2,5 |

laozhang.ai — реселлер со скидкой к официальным ценам, реальная маржа выше.
Бонусы тарифов (+14…+41%) размывают маржу пропорционально — худший кейс
(Макс, только Opus): ×2,8 / 1,41 ≈ ×2 — приемлемо.

Минимальное списание: `MIN_CHARGE_KOPECKS=10` (0,10 ₽) — защита от
микросписаний при токенном dev-API.

## Риски / заметки

- Цены моделей применяются повторным запуском `add_laozhang_models`
  (команда перезаписывает `cost_kopecks` — ручные правки в админке будут затёрты).
- `setup_tariffs` деактивирует прочие активные платные тарифы
  (флаг `--keep-others` отключает).
- XTR-пакеты бота (100 ₽/50 XTR и т.д.) не меняли: нетто-выручка ~50–70 ₽
  со 100 ₽ баланса, при новых ценах маржа сохраняется.
- При росте курса $ или цен laozhang пересмотреть Opus (маржа ×2,8 — самая тонкая).
