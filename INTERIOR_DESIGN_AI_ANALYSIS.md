# AI Дизайн интерьера — Анализ для aineron.ru

> Дата: 2026-06-29  
> Методология: рыночное исследование через WebSearch (Opus 4.8) + анализ стека aineron.ru  
> Все рыночные данные — подтверждены первичными источниками (similarweb, semrush, levelsio/X, pricing pages)

---

## TL;DR — Стоит ли делать?

**Да, но не как отдельный продукт — как вертикальный режим поверх уже существующего стека.**

Базовая ниша «фото → рестайлинг за рубли без VPN» уже занята. Незакрытая возможность — специализированный UX (пресеты стилей, комнат, задач) + первый RU-сервис с реальными товарами из российских каталогов (Hoff, Lazurit, Leroy Merlin). Стек aineron.ru позволяет сделать конкурентоспособный v1 без покупки новых моделей.

## 3. РЫНОЧНЫЙ СПРОС

### Глобально
| Сегмент | Размер 2024/25 | CAGR | Прогноз |
|---|---|---|---|
| Interior Design Software (широко) | $5.3–6.8B | 7–11% | ~$10–13B к 2031 |
| **AI in Interior Design (нишевый)** | **$1.4–1.5B** | **21–27%** | **~$5–7B к 2030–32** |

### В России — важный контекст
**Спрос на дизайн интерьера упал на 20–40% г/г в Q4 2025** из-за высоких ипотечных ставок (25%+) и отложенных ремонтов (Яндекс.Вордстат, basicdecor.ru). Это временный фактор — спрос вернётся при снижении ставки.

### Целевые сегменты (по объёму)
1. **Домовладельцы** — "хочу увидеть как будет выглядеть до ремонта" — самый массовый
2. **Профессиональные дизайнеры** — ~67% уже используют AI для концептов; платят больше, ценят качество и итерационность

## 4. ТЕХНОЛОГИЧЕСКИЙ АНАЛИЗ

### Что используют конкуренты
- **RoomGPT (open source):** SD + ControlNet MLSD — извлекает карту линий из фото, SD генерирует новый стиль сохраняя геометрию
- **InteriorAI:** Flux (Pro) + проприетарная "Hyper Realism" модель
- **Reimagine Home:** Вероятно segmentation + inpaint (замена отдельных элементов)

### Специализированные interior-модели (существуют!)
- **XSarchitectural LoRA** (Civitai) — SD LoRA с 10+ стилями: Modern, Japanese, Bedroom и т.д.
- **Interior-Flux-Lora** (HuggingFace: SedatAl) — LoRA для FLUX.1-dev, trigger word "interior"
- **fal.ai: flux-2-lora-gallery/apartment-staging** — FLUX.2 LoRA для staging мебели

### Стек aineron.ru — что уже есть

| Что нужно | Что есть у aineron | Готово? |
|---|---|---|
| Photo → restyle | Flux Kontext Pro + img2img | ✅ |
| Замена отдельных элементов | MaskEditor + GPT Image inpaint | ✅ |
| Расширение изображения | Outpaint + Generative Expand | ✅ |
| Библиотека стилей | /prompts/ (промпт-шаблоны) | ✅ (нужен UX) |
| Пресеты настроек strength/guidance | img2img параметры | ✅ (нужна преднастройка) |
| 2D план → 3D рендер | **Нет** (нужен ControlNet/depth) | ❌ |
| RU мебельные каталоги | **Нет** | ❌ (возможность!) |

### Flux Kontext Pro для интерьеров
Ключевое свойство: **сохранение структуры/композиции при целевых правках** ("targeted modifications of specific elements without affecting the rest… while maintaining the overall composition" — Together AI). Это именно то, что нужно для редизайна: "оставь геометрию комнаты, поменяй стиль на Scandinavia".

Сравнение с ControlNet: Kontext сохраняет геометрию через промпт-контроль (не через явную MLSD карту) — немного менее жёсткая фиксация, но практически достаточно для большинства задач.

**Вывод: aineron.ru уже имеет всё необходимое для конкурентоспособного v1 без покупки новых моделей.**

## 5. АНАЛИЗ ЦЕЛЕСООБРАЗНОСТИ

### Tier 1 — Строится сейчас, без новых моделей (РЕКОМЕНДУЮ)

**Что делаем:** специализированный UX поверх существующего стека

- Раздел "Дизайн интерьера" в чате/каталоге
- Библиотека промпт-шаблонов по задачам:
  - По стилю: «Скандинавский», «Минимализм», «Лофт», «Japandi», «Классика», «Современный», «Бохо»
  - По комнате: «Гостиная», «Спальня», «Кухня», «Детская», «Ванная», «Коридор»
  - По задаче: «Перекрасить стены», «Заменить мебель», «Добавить освещение», «Перепланировка без сноса»
- Преднастроенные img2img параметры (strength ~0.65 для интерьеров — сохраняет геометрию)
- MaskEditor предзагружен для "замени только диван"
- Бейдж «работает без VPN, оплата в рублях»

**Трудоёмкость:** 1–2 дня (Frontend UX + промпт-библиотека)  
**Новые модели:** не нужны  
**Новые провайдеры:** не нужны

### Tier 2 — Дифференциаторы (делаем после проверки спроса)

| Фича | Что даёт | Трудоёмкость | Приоритет |
|---|---|---|---|
| **RU мебельные каталоги** (Hoff, Lazurit, Leroy Merlin) | Первый в РФ shoppable interior design | 2–3 нед | Высокий |
| **Спецификация** — список мебели с размерами после рендера | Закрывает главную боль дизайнеров | 1–2 нед | Высокий |
| **Итерационные правки** — маска + переспросить только про зону | Конкурирует с Reimagine Home | 3–5 дн | Средний |
| Interior-Flux-Lora подключение | Качество стилей как у InteriorAI | 1–2 дн | Средний |

### Tier 3 — Не делаем сейчас

- **2D план → меблированный 3D рендер** — нужен ControlNet/depth conditioning, которого нет на laozhang.ai. Требует fal.ai или Replicate. Это то, чего хотел бы пользователь, но реализация сложна и результат непредсказуем без fine-tuning.
- **360° туры / VR** — уровень InteriorAI Ultra, требует отдельного рендер-движка.

| Согласованность стиля между ракурсами | Никто |
| Нет photo-redesign у RU Big Tech (Kandinsky, Шедеврум) | **Aineron.ru может закрыть** |

## 7. РИСКИ

| Риск | Вероятность | Митигация |
|---|---|---|
| RU спрос упал из-за ипотеки | Высокая (подтверждено) | Ориентироваться на дизайнеров (B2B), не только на домовладельцев |

| RoomGPT.ru добавит те же фичи | Средняя | Скорость запуска + качество промптов |
| Ипотечный кризис затянется | Средняя | Параллельно развивать B2B-дизайнеров |

## 8. РЕКОМЕНДУЕМЫЙ ПЛАН

### Шаг 1 (1–2 дня) — Проверка спроса без разработки
Добавить в существующий чат с Flux Kontext Pro:
- Закреплённый промпт: `"Redesign this room in [STYLE] style. Keep the room geometry, windows, and doors exactly as they are. Change only furniture, colors, and decorations."`
- Объявить в TG-боте: «AI Дизайн интерьера — загрузи фото комнаты»
- Замерить конверсию за 2 недели

### Шаг 2 (3–5 дней) — Специализированный UX
- Раздел «Дизайн интерьера» с пресетами стилей и комнат
- Страница `/interior/` с landing (SEO: «AI дизайн интерьера онлайн без VPN»)
- Strength=0.65 по умолчанию для interior img2img

### Шаг 3 (2–3 недели) — Дифференциатор
- Интеграция с Hoff API или парсинг каталога
- После рендера: «Найти похожую мебель» → карточки с ценами
