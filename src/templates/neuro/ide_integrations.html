Инструкция

Cursor

Cursor поддерживает кастомные OpenAI-провайдеры начиная с версии 0.40. Настройка занимает меньше минуты.

Шаг 1. Откройте настройки модели

Нажмите Ctrl+Shift+J (или Cmd+Shift+J на Mac) → вкладка Models → кнопка Add Model.

Шаг 2. Добавьте кастомный провайдер

Прокрутите вниз до раздела OpenAI API Key, затем нажмите Override OpenAI Base URL и введите:

https://aineron.ru/api/v1

В поле API Key введите ваш ключ ak_....

Шаг 3. Укажите модель

В поле Model Name введите идентификатор нужной модели, например gpt-4o или claude-3-5-sonnet-20241022. Посмотреть все доступные модели можно через:

curl https://aineron.ru/api/v1/models \
  -H "Authorization: Bearer ak_ВАШ_КЛЮЧ"

Шаг 4. Сохраните и проверьте

Нажмите Save. В правом нижнем углу редактора выберите добавленную модель и откройте чат (Ctrl+L). Отправьте тестовое сообщение — готово.

Cursor кэширует ответы для автодополнения. При нехватке звёзд автодополнение временно отключится — пополните баланс в личном кабинете.

Cline

Cline (бывший Claude Dev) — расширение VS Code, которое может самостоятельно создавать и редактировать файлы, запускать команды и серфить веб. Поддерживает любой OpenAI-совместимый провайдер.

Шаг 1. Установите расширение

Найдите Cline в маркетплейсе VS Code (расширений) и установите. После этого появится иконка в левой панели.

Шаг 2. Настройте провайдер

Откройте настройки Cline (иконка шестерёнки) → раздел API Provider → выберите OpenAI Compatible.

Заполните поля:

API Base URL:  https://aineron.ru/api/v1
API Key:       ak_ВАШ_КЛЮЧ
Model ID:      gpt-4o

Шаг 3. Выберите модель

В поле Model ID введите любой идентификатор из нашего каталога. Рекомендованные для агентских задач:

  • gpt-4o — универсальный выбор, хорошо следует инструкциям
  • claude-3-5-sonnet-20241022 — лучший для работы с кодом и длинными файлами
  • claude-opus-4-5 — максимальные возможности для сложных задач

Шаг 4. Первая задача

Откройте любой проект, нажмите на иконку Cline в боковой панели и опишите задачу на русском или английском. Cline попросит разрешение перед каждым изменением файла.

Агентские сессии Cline могут расходовать больше токенов, чем обычный чат, так как модель читает контекст файлов. Следите за балансом через личный кабинет.

Continue

Continue — открытый AI-ассистент для VS Code и JetBrains с гибкой конфигурацией провайдеров через JSON-файл.

Шаг 1. Установите расширение

Установите Continue из маркетплейса VS Code или JetBrains. После установки откройте файл конфигурации: ~/.continue/config.json (или через команду Continue: Open Config).

Шаг 2. Добавьте провайдер

Отредактируйте config.json, добавив блок models:

{
  "models": [
    {
      "title": "GPT-4o (aineron.ru)",
      "provider": "openai",
      "model": "gpt-4o",
      "apiBase": "https://aineron.ru/api/v1",
      "apiKey": "ak_ВАШ_КЛЮЧ"
    },
    {
      "title": "Claude 3.5 Sonnet (aineron.ru)",
      "provider": "openai",
      "model": "claude-3-5-sonnet-20241022",
      "apiBase": "https://aineron.ru/api/v1",
      "apiKey": "ak_ВАШ_КЛЮЧ"
    }
  ],
  "tabAutocompleteModel": {
    "title": "GPT-4o Mini (aineron.ru)",
    "provider": "openai",
    "model": "gpt-4o-mini",
    "apiBase": "https://aineron.ru/api/v1",
    "apiKey": "ak_ВАШ_КЛЮЧ"
  }
}

Шаг 3. Активируйте

Сохраните файл — Continue автоматически применит новую конфигурацию. В выпадающем меню в правом нижнем углу чата выберите нужную модель.

Горячие клавиши Continue

СочетаниеДействие
Ctrl+LОткрыть чат / добавить выделение в контекст
Ctrl+IРедактировать выделенный код
TabПринять автодополнение
Ctrl+Shift+RВыбрать другую модель
Continue поддерживает несколько моделей одновременно — вы можете переключаться между GPT-4o и Claude, не меняя конфигурацию.

Советы и типичные вопросы

Какую модель выбрать для кода?

Для большинства задач: claude-3-5-sonnet-20241022 или gpt-4o. Для быстрого автодополнения подойдёт gpt-4o-mini — он дешевле и значительно быстрее.

Как проверить остаток звёзд?

Баланс отображается в заголовке личного кабинета на сайте или через эндпоинт GET /api/v1/keys/ (поле balance_stars в будущем обновлении).

Что делать при ошибке 429?

Превышен лимит 120 запросов в минуту на ключ. IDE обычно автоматически повторяет запрос через несколько секунд. Если проблема постоянная — создайте несколько ключей для разных инструментов.

Работает ли автодополнение?

Да, Cursor и Continue используют эндпоинт /chat/completions для автодополнения. Рекомендуем назначить для него быструю модель (gpt-4o-mini), чтобы снизить задержку и расход звёзд.
